# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts house prices using various Machine Learning regression models.  
The dataset is analyzed, preprocessed, and multiple algorithms are applied to compare performance and improve prediction accuracy.

The goal is to estimate the selling price of houses based on important features such as area, quality, number of bedrooms, garage capacity, and more.

---

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

---

## 📊 Features Implemented
✔ Data loading and preprocessing  
✔ Exploratory Data Analysis (EDA)  
✔ Train-Test Split  
✔ Simple Linear Regression  
✔ Multiple Linear Regression  
✔ Ridge Regression  
✔ Lasso Regression  
✔ Random Forest Regression  
✔ Model Evaluation using:
- Mean Squared Error (MSE)
- R² Score  
✔ Visualization of:
- Price vs Area
- Actual vs Predicted Prices

---

## 🧠 Machine Learning Models Used

1. **Linear Regression**
2. **Ridge Regression**
3. **Lasso Regression**
4. **Random Forest Regressor**

Random Forest model generally provides better performance compared to basic linear models.
